1. Install Python in your computer
2. Unzip the WebSearchEngineV3.zip
3. Open a command line prompt(For Windows: Press Win + R, type "cmd", and press Enter; For MacOS: Press Command + Space, type "Terminal", and press Enter) 
4. Go to the directory WebSearchEngine
5. Run the python program in command line: python indexer.py   (index will be saved in file index.json)
6. Run the python program in command line: python searcher.py   (ranking result is stored in output.txt)
